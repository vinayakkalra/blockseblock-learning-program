# Buy Me a Coffee Hardhat Project

This project demonstrates a basic "Buy Me a Coffee" smart contract along with test cases. It includes a sample contract, tests for the contract, and a Hardhat Ignition module for deployment.

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
npx hardhat ignition deploy ./ignition/modules/Coffee.ts
```

To deploy on sepolia, uncomment and add alchemy api key and your private key in hardhat.config.json file. Run the below command to deploy on sepolia network

```shell
npx hardhat ignition deploy ./ignition/modules/Coffee.ts --network sepolia
```
